def standard_list(list_to_standardize):
    """
    created by deepak
    description the method taked the list and send the list after standardizing it by performing the vectorized operations to it.
    """
    mean=list_to_standardize.mean()
    sd=list_to_standardize.std()
    return (list_to_standardize-mean)/sd

Hypothesis Testing of popularity of the action movie¶

Here in this hypothesis test we will try to prove the null hypothesis wrong that the vote average of action movies is different from the normal movies by performing the two tail z-test on our data set. The sample size of our data set is more than 30 and we will assume that we are considering the total population and not the sample data of the movies.

Null Hypothesis (H0): The vote average of the action movies is same of that of the all movies. Alternate Hypothesis(HA): The vote average of the action movies and other movies are not same.

Test type - A two tail z-test is performed for the analysis for 95% confidence inteval.

Action movie data function¶

The function takes the dataframe as input and returns the dataframe where the genre of the movie contains action on it.

Data wrangling and cleaning process¶

  • Here we are creating a new empty dataframe as action_df which will have the vote_average in it.
  • We have uses a for loop to iterate to get the genres string.
  • We are checking if the values of the genre string is not NaN with the if condition.
  • The genre string contains the movie genre separated by |.
  • We have separated the multiple genres with the split method which returns the array of genres.
  • Then we are iterating over the one dimension array using the list to search for the action word.
  • If the array contains the action we are adding the average vote to the action_df dataframe
  • In the end we are returning the action_df dataframe.
In [249]:
def action_movie_list(genre_df): 
    """
    created by deepak
    description funtion for returning the vote average dataframe from the given dataframe
    limitation this method assumes that the dataframe contains the vote_average column and returns only the action genres vote average.
    also the loop are used as the appropriate built in funcions were not present.
    """
    action_df = pd.DataFrame(columns=['vote_average'])
    for index, row in genre_df.iterrows():        
        genre_string=row['genres']               
        if genre_string!=NaN:
            movie_list=np.array(genre_string.split('|'))
            for i in np.nditer(movie_list):
                if(i=='Action'):
                    action_df=action_df.append({'vote_average':row['vote_average']}, ignore_index=True) 

    return action_df

Data Wrangling for hypothesis test¶

  • The genre_df dataframe contains the rows only where the genres string column is not empty and thus eliminating the empty data.
  • The action_movie_df contains the new dataframe containing the vote average of only the action movies.
  • We are calculation the mean,standard deviation of all the movies and the mean of the action list by taking the vote_average column from the data frame and converting it to the one dimensional list.
  • We then calculated the z-score of the analysis.
